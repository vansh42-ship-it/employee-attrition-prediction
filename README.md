# Employee Attrition Prediction System

## 🚀 Overview
This project predicts whether an employee is likely to leave the company using Machine Learning.

It is built using:
- Random Forest Classifier
- IBM HR Analytics Dataset
- FastAPI backend
- React frontend

---

## 🧠 Features
- Predict employee attrition in real-time
- Handles class imbalance using resampling
- Feature engineering for better accuracy
- Full-stack integration (React + FastAPI)

---

## ⚙️ Tech Stack
- Python (scikit-learn, pandas)
- FastAPI
- React (TypeScript)
- GitHub

---

## 🔍 How it works
1. User enters employee details
2. Frontend sends data to backend API
3. ML model processes input
4. Prediction is returned (Leave / Stay)
5. Results displayed in UI

---

## 📊 Model Improvements
- Balanced dataset using upsampling
- Added features like:
  - LowIncome
  - HighRiskCombo
  - ExtremeStress
- Tuned threshold for better recall

---

## 🧪 Example Output
- Attrition Risk: 65%
- Status: Likely to Leave
